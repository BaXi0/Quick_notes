# Quick_notes
Prosta aplikacja do tworzenia szybkich i krótkich notatek z datą i godziną.

## Funkcjonalności:
  - dodawanie notatek z aktualną datą, godziną;
  - dodawanie notatek z własną datą, godziną;
  - przeglądanie utworzonych notatek;
  - wyczyszczenie pliku z notatkami;
  - zapisywanie danych do pliku;
  - proste i czytelne menu w terminalu.

## Technologie:
  - Python 3;
  - SQLite;
  - JSON

## Plany rozwoju:
  - zapisywanie notatek w bazie danych SQLite;
  - dodanie pliku konfiguracyjnego
  - interfejs webowy;
  - logowanie użytkowników;
  - wyszukiwanie notatek.
